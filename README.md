## 프로젝트 소개
- [브릿지AI 정부지원 캘린더](https://www.bridgeai.co.kr/)는 성공적인 스타트업 운영을 위한 정부지원사업 • 정책자금 알리미 서비스입니다.
- 지원사업 캘린더와 리스트 UI를 통해 나에게 딱 맞는 지원사업을 빠르게 찾아 보세요.
- 쉽고 간단한 UI로 AI 사업계획서 작성 서비스를 이용해 보세요.
